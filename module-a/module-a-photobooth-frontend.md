## 개요

누구나 손쉽게 사진을 찍고 꾸며 출력할 수 있는 포토부스 키오스크 서비스인 PhotoBooth의 레이아웃 및 프론트엔드 화면을 구현한다. 본 과제는 MVP(Minimum Viable Product) 수준의 구현을 목표로 하며, 실제 카메라 연결은 고려하지 않는다. 촬영 화면에서의 사진은 제공된 `media` 폴더 내 샘플 이미지를 사용하여 구현한다.

---

## 공통 사항

- 모든 화면은 한국어/영어 전환을 지원하며, 헤더의 토글 버튼으로 전체 화면의 텍스트를 전환한다.
- 본 문서에서 UI 텍스트는 `한국어 / English` 형식으로 표기한다.

---

## 1. 시작 화면

서비스의 진입점이 되는 화면으로, PhotoBooth 로고와 브랜드 문구를 표시한다. `터치하여 시작 / Touch to Start` 문구는 깜빡이는 애니메이션을 적용하며, 화면의 아무 곳이나 클릭하면 프레임 선택 화면으로 이동한다. 단, 한/영 토글 버튼 클릭은 화면 이동에서 제외된다. 한/영 토글을 통해 전체 화면의 UI 텍스트를 전환할 수 있으며, 이는 모든 화면에 공통으로 적용된다.

---

## 2. 프레임 선택 화면

사용자가 촬영할 네컷 사진의 레이아웃을 선택하는 화면이다. 제공된 `frame_icon_*.png` 3개를 카드 형태로 표시하며, 카드를 클릭하면 선택 상태를 시각적으로 표시한다. 선택한 프레임 타입은 이후 촬영, 편집, 결과 화면의 사진 레이아웃에 공통으로 반영된다. `시작하기 / Start` 버튼은 프레임을 선택하기 전까지 비활성화 상태로 표시되며, 프레임 선택 후 활성화된다. 버튼을 클릭하면 촬영 화면으로 이동한다.

| 프레임 | 타입 | 레이아웃 |
|---|---|---|
| frame_icon_01 | 세로 4컷 / Vertical 4-cut | 1열 4행 |
| frame_icon_02 | 가로 4컷 / Horizontal 4-cut | 4열 1행 |
| frame_icon_03 | 2x2컷 / 2x2-cut | 2열 2행 |

---

## 3. 촬영 화면

선택한 프레임 타입에 따라 샘플 이미지를 촬영하는 화면이다. 카메라 영역은 빈 영역으로 표시하며, 5초 카운트다운 숫자를 표시한다. 카운트다운 종료 시 자동으로 촬영되며, `촬영 / Shoot` 버튼을 클릭하면 카운트다운과 관계없이 즉시 촬영된다. 1장 촬영 완료 후 카운트다운은 다시 5초부터 시작된다. 촬영 시 전체 화면이 흰색으로 번쩍이는 플래시 효과를 표시하고, `media` 폴더의 샘플 이미지를 사용하여 랜덤으로 슬롯에 순서대로 채워진다. 4장 촬영 완료 시 자동으로 편집 화면으로 이동한다.

샘플 이미지는 선택한 프레임 타입에 따라 아래와 같이 구분하여 사용한다.

| 프레임 타입 | 샘플 이미지 |
|---|---|
| 세로 4컷, 가로 4컷 | `sample_square_*.png` 또는 `sample_horizontal_*.png` 랜덤 |
| 2x2컷 | `sample_vertical_*.png` 랜덤 |

---

## 4. 편집 화면

촬영된 사진을 프레임에 배치하고 꾸미는 화면으로, 아래 3단계로 진행된다. 각 단계는 `다음 / Next` 버튼을 클릭하여 다음 단계로 이동한다.

### Step 1. 사진 배치 / Photo Arrangement

촬영된 사진 4장이 슬롯에 표시된다. 슬롯에서 사진을 클릭하면 선택한 프레임 타입에 맞는 프레임 이미지(`frame_icon_01.png` → `frame_01.png`, `frame_icon_02.png` → `frame_02.png`, `frame_icon_03.png` → `frame_03.png`) 위의 빈 칸 중 가장 앞 칸부터 순서대로 채워지며, 같은 사진을 중복으로 선택할 수 있다. 프레임에 배치된 사진을 클릭하면 해당 사진이 빠지고 빈 칸으로 돌아간다. 4장이 모두 채워지면 `다음 / Next` 버튼이 활성화된다.

### Step 2. 필터 적용 / Filter

프레임에 배치된 사진에 필터를 적용한다. 선택 가능한 필터는 `기본 / Default` 과 `흑백 / B&W` 이며, 초기 선택 상태는 `기본 / Default` 이다. 필터를 선택하면 프레임 미리보기에 실시간으로 반영된다. `다음 / Next` 버튼을 클릭하여 다음 단계로 이동한다.

### Step 3. 스티커 적용 / Sticker

`sticker_01.png` ~ `sticker_04.png` 중 스티커를 선택한 후 프레임 위 원하는 위치를 클릭하면 해당 위치에 스티커가 삽입된다. 스티커의 초기 크기는 100x100px이며, 크기를 조절할 수 있다. 스티커는 중복으로 삽입할 수 있으며, 삽입된 스티커를 삭제할 수 있다. 모든 편집이 완료되면 `다음 / Next` 버튼을 클릭하여 결과 화면으로 이동한다.

---

## 5. 결과 화면

편집이 완료된 최종 프레임 사진을 표시하는 화면이다. `JPG 다운로드 / Download JPG` 버튼과 `PNG 다운로드 / Download PNG` 버튼을 클릭하면 해당 형식의 파일로 실제 저장된다. `홈으로 / Home` 버튼을 클릭하면 시작 화면으로 이동한다.
