## 개요

뮤지컬, 가수 공연 등 다양한 공연의 티켓을 예매할 수 있는 서비스인 TicketBox의 백엔드를 구현한다. 제공된 레이아웃을 활용하여 구현할 수 있으며, 직접 제작하여도 무방하다.

---

## 공통 사항

- `data.json`의 공연 데이터가 모두 데이터베이스에 삽입되어 있어야 한다.
- 좌석은 행(A, B, C ...)과 열(1~10)으로 구성되며, 총 좌석 수에 따라 행 수가 자동으로 결정된다. (예: 40석 → A~D행, 50석 → A~E행)
- 네비게이션은 로그인 상태에 따라 아래와 같이 구성된다.

| 상태 | 메뉴 |
|---|---|
| 비로그인 | 로고, 공연 목록, 로그인, 회원가입 |
| 로그인 | 로고, 공연 목록, 예매 내역, 회원 이름, 로그아웃 |

---

## 1. 회원가입 / 로그인

### 회원가입

이메일, 비밀번호, 이름, 전화번호를 입력받는다. 아래 유효성 검사를 적용한다.

- 이메일 형식 검사 및 중복 가입 방지
- 비밀번호 최소 4자 이상
- 전화번호는 숫자만 입력 가능

비밀번호 입력 필드는 표시/숨김 토글(`type=password` ↔ `type=text`)을 지원한다. 회원가입 완료 시 가입한 계정으로 자동 로그인되며 공연 목록 조회 페이지로 이동한다.

### 로그인

이메일과 비밀번호로 로그인한다. 로그인 완료 시 공연 목록 조회 페이지로 이동한다.

### 공통

로그인 상태에서는 모든 페이지에서 로그인·회원가입 버튼이 표시되지 않으며, 회원 이름과 로그아웃 버튼이 표시된다. 로그아웃 버튼을 클릭하면 로그아웃 처리 후 공연 목록 조회 페이지로 이동한다.

---

## 2. 공연 목록 조회

공연 목록을 표시한다. 각 공연은 공연명, 날짜/시간, 장소, 포스터 이미지, 티켓 가격, 잔여석/총좌석을 표시한다. 공연명 검색, 장소별 필터, 날짜순 정렬(오름차순/내림차순) 기능을 제공한다. 날짜순 정렬의 기본값은 내림차순이다. 공연을 클릭하면 해당 공연의 상세 조회 페이지로 이동한다.

---

## 3. 공연 상세 조회

공연의 상세 정보를 표시한다. 공연명, 날짜/시간, 장소, 포스터 이미지, 티켓 가격, 잔여석/총좌석, 공연 설명을 표시한다. 예매하기 버튼을 표시하며, 잔여석이 없을 경우 예매하기 버튼은 비활성화된다. 예매하기 버튼을 클릭하면 티켓 예매 페이지로 이동한다.

---

## 4. 티켓 예매

좌석 배치도를 표시하고 좌석을 선택하여 예매하는 페이지다. 아래 조건에 따라 페이지 접근을 제한한다.

- 비로그인 상태로 접근 시 로그인이 필요하다는 메시지를 표시하고 로그인 페이지로 이동한다.
- 공연 날짜가 이미 지난 경우 종료된 공연임을 알리는 메시지를 표시하고 해당 공연의 상세 조회 페이지로 이동한다.
- 잔여석이 없는 경우 매진임을 알리는 메시지를 표시하고 해당 공연의 상세 조회 페이지로 이동한다.

좌석 배치도는 총 좌석 수에 맞게 행(A, B, C ...)과 열(1~10)로 구성되며, 예매된 좌석과 예매 가능한 좌석을 시각적으로 구분하여 표시한다. 이미 예매된 좌석은 비활성화된다. 티켓 가격은 좌석당 5,000원이다.

좌석은 클릭하여 선택하고, 다시 클릭하여 선택을 취소할 수 있으며, 여러 좌석을 자유롭게 선택할 수 있다. 좌석을 선택하지 않은 경우 예매 확정 버튼은 비활성화된다. 좌석 선택 후 예매 확인 단계에서 선택한 좌석 정보와 총 가격을 표시하며, 예매 확정 시 예매가 완료된다.

예매 확정 시 이미 다른 사용자가 선택한 좌석과 중복될 경우, 중복된 좌석 정보를 (예: A3, B6) 알리는 메시지를 표시하고 티켓 예매 페이지로 이동한다.

---

## 5. 예매 내역 조회

로그인한 회원의 예매 내역을 테이블로 표시한다. 비로그인 상태로 접근 시 로그인이 필요하다는 메시지를 표시하고 로그인 페이지로 이동한다. 테이블에는 공연명, 날짜/시간, 장소, 예매 좌석 수, 결제 금액, 예매 일시를 표시한다. 예매 완료 후 자동으로 이동한 경우 방금 예매한 내역의 row를 하이라이트 처리하여 표시한다. row를 클릭하면 예매 상세 페이지로 이동한다. 각 row에는 취소 버튼을 표시하며, 취소 버튼 클릭 시 해당 예매가 취소된다. 단, 공연 날짜가 이미 지난 경우 취소 버튼을 표시하지 않는다.

---

## 6. 예매 상세

예매 내역의 상세 정보를 입장권 UI로 표시한다. 예매한 좌석 수만큼 입장권을 표시하며, 각 입장권에는 좌석 번호 하나씩 표시된다. 입장권에는 공연명, 날짜/시간, 장소, 좌석 번호, 좌석당 가격(5,000원), 예매 일시를 표시한다. 뒤로가기 버튼을 클릭하면 예매 내역 조회 페이지로 이동한다.
